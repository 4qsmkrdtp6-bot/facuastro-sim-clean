# 🌌 NASA Style Solar System Simulator

Simulador 3D del sistema solar estilo NASA Eyes usando Three.js.

---

## 🚀 Características

- Sistema solar 3D básico
- Cámara libre tipo exploración espacial
- Órbita de la Tierra alrededor del Sol
- Render WebGL en tiempo real
- Controles de mouse (OrbitControls)

---

## 🧠 Tecnología

- Three.js (WebGL)
- JavaScript ES Modules
- OrbitControls (cámara interactiva)

---

## 📁 Estructura

index.html → entrada principal  
main.js → simulación 3D

---

## 🚀 Cómo ejecutar

Solo abre el proyecto en un servidor o despliega en Vercel.

En GitHub + Vercel:

1. Subir ambos archivos a la raíz
2. Deploy automático en Vercel
3. Abrir URL generada

---

## 🎮 Controles

- Mouse: rotar cámara
- Scroll: zoom
- Arrastrar: mover vista
